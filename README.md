# MOTORA
![Motora HomepagePreview](https://raw.githubusercontent.com/Md-Abdullah303/motora-client/main/public/MOTORA.png)

MOTORA is a modern and elegant premium car marketplace built with Next.js. It is designed to showcase a polished online high-performance vehicle experience with a beautiful UI, smooth presentation, and a strong foundation for future product pages, real-time AI capabilities, and user interaction.

## 📌 Project Name
MOTORA – Next-Generation Agentic AI Marketplace

## 🎯 Purpose
This project is a fully functional, elegant platform dedicated to hypercars and luxury vehicles. It is designed to provide a premium user experience from browsing to checkout, featuring secure authentication, robust state management, AI-driven recommendations, and a user-centric dashboard. It serves as a complete, modern web application ready for real-world usage.

## ✨ Key Features
- **User Authentication & Profiles:** Secure login, registration, and profile management with avatar uploads and Google Auth (via better-auth).
- **Dynamic Vehicle Browsing:** Shop by category, view detailed vehicle pages with high-quality images, specifications, and related vehicles.
- **Agentic AI Features:**
  - *AI Auto-Classification:* Smart neural engine to auto-generate full technical descriptions from raw text in the Dashboard.
  - *AI Chat Assistant:* A 24/7 AI-driven chatbot powered by Gemini that answers queries and helps users find the perfect car.
- **Secure Stripe Checkout:** Integrated real-time payments with Stripe Elements for a safe and smooth checkout experience.
- **User Dashboard:** Dedicated dashboard for users to add and manage their listed cars and view their personal analytics and payment history.
- **Premium UI/UX:** Fully mobile responsive, accessible, and beautifully themed dark mode interface using Tailwind CSS. Ensures flawless experience across desktop, tablet, and mobile devices.

## 🧰 Technologies Used

**Frontend**
- **Framework:** Next.js (App Router), React, TypeScript
- **Styling & UI:** Tailwind CSS, Recharts, Lucide Icons
- **State & Data Fetching:** React Hooks, native fetch API
- **Payments:** Stripe.js
- **Authentication:** better-auth Client

**Backend / Services**
- **Server:** Node.js, Express.js, TypeScript
- **Database:** MongoDB (Mongoose)
- **AI Integrations:** Google Gemini AI API
- **Payments:** Stripe Node.js SDK
- **Authentication:** better-auth Server with custom JWT validation
- **Image Hosting:** ImgBB API
